These are procedurally generated by `scripts/generate-sounds.mjs` (plain sine/triangle synthesis written straight to WAV, no external assets/codec needed):

- `tick.wav` — reel passing an item during a spin (fires rapidly, short/punchy)
- `win.wav` — case win at Normal/Selten/Mythisch rarity
- `ultra-win.wav` — case win at Ultra rarity (plays together with the confetti burst)
- `click.wav` — generic UI feedback (case buttons, wardrobe filters, Anlegen/Ablegen)
- `error.wav` — failed action (not enough credits, etc.)
- `flip.wav` — Double-or-Nothing flip start
- `save.wav` — confirmation chime for any successful save (admin editors, account/profile settings)
- `hover.wav` — subtle hover feedback on buttons/icons/nav (fires often, quiet, <100ms)
- `hit.wav` — melee swing landing on a target in the 3D World (interrupt channel, never queues, so rapid swings each play immediately)

To regenerate (e.g. after tweaking the synthesis parameters): `node scripts/generate-sounds.mjs`.

If you'd rather swap these for real recorded/licensed SFX later, just drop same-named files of any browser-supported format here and update the extension in `lib/sound-manager.ts`'s `FX_SRC`/`INTERRUPT_SRC` maps.
